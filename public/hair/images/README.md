# Картинки для приложения парикмахерской

## Необходимые файлы

### Галерея работ (work1.jpg - work4.jpg)
- Размер: 300x300px (квадрат)
- Формат: JPG или PNG
- Примеры: фото работ мастеров, результаты стрижек, окрашиваний, маникюра

### Услуги (service_*.jpg)
- service_haircut.jpg — фото стрижки
- service_color.jpg — фото окрашивания
- service_nails.jpg — фото маникюра
- service_styling.jpg — фото укладки
- Размер: 200x200px (квадрат)

### Мастера (master_*.jpg)
- master_anna.jpg — фото Анны
- master_dmitry.jpg — фото Дмитрия
- master_elena.jpg — фото Елены
- Размер: 200x200px (квадрат, портрет)

### Заглушка (placeholder.jpg)
- Показывается, если основная картинка не загрузилась
- Размер: любой

## Где взять картинки?

1. **Бесплатные стоки:**
   - Unsplash: https://unsplash.com/s/photos/hair-salon
   - Pexels: https://www.pexels.com/search/hair%20salon/
   - Pixabay: https://pixabay.com/images/search/salon/

2. **Свои фото:**
   - Сфотографируйте реальные работы мастеров
   - Сделайте портреты мастеров
   - Обрежьте до нужного размера
   - Оптимизируйте через TinyPNG: https://tinypng.com/

## Оптимизация

Перед загрузкой обязательно сожмите картинки:
- Онлайн: https://tinypng.com/
- Офлайн: ImageOptim (Mac), FileOptimizer (Windows)

Цель: каждая картинка < 100KB
