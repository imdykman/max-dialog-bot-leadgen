# Картинки для приложения

## Необходимые файлы

Загрузите в эту папку следующие картинки:

### Услуги (service1.jpg - service4.jpg)
- Размер: 300x200px
- Формат: JPG или PNG
- Примеры: фото услуг, работ, интерьеров

### Специалисты (specialist1.jpg - specialist3.jpg)
- Размер: 200x200px (квадрат)
- Формат: JPG или PNG
- Примеры: фото специалистов, аватарки

### Заглушка (placeholder.jpg)
- Размер: любой
- Формат: JPG или PNG
- Показывается, если основная картинка не загрузилась

## Где взять картинки?

1. **Бесплатные стоки:**
   - Unsplash: https://unsplash.com/
   - Pexels: https://www.pexels.com/
   - Pixabay: https://pixabay.com/

2. **Свои фото:**
   - Сфотографируйте реальные услуги и специалистов
   - Обрежьте до нужного размера
   - Оптимизируйте через TinyPNG: https://tinypng.com/

## Оптимизация

Перед загрузкой обязательно сожмите картинки:
- Онлайн: https://tinypng.com/
- Офлайн: ImageOptim (Mac), FileOptimizer (Windows)

Цель: каждая картинка < 100KB
